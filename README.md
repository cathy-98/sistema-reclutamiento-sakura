# sistema-reclutamiento-sakura
Este repositorio contiene el código fuente unificado para el módulo de autenticación (Login) del Sistema de Reclutamiento y Evaluación de Elitsoft. El proyecto está estructurado de manera monorrepisitorio local, separando limpiamente el ecosistema del servidor, la interfaz del usuario y la orquestación del ambiente a través de contenedores.
